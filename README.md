# PW Downloader Telegram Bot

This is a Physics Wallah (PW) video downloader Telegram bot with proxy support.

## Features

- Download PW videos
- Proxy support
- Auto detect links
- Fast yt-dlp downloader
- Render deploy ready

## Commands

/start - Start bot  
/stop - Stop download  

## Deploy

1. Upload files to GitHub
2. Connect GitHub to Render
3. Add environment variables:
   - API_ID
   - API_HASH
   - BOT_TOKEN
   - OWNER
   - PUBLIC_URL

4. Deploy and run
